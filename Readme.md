# Personal Information Program

## Description
A simple Python program that stores personal information and collects user preferences from the user. It demonstrates variables, user input, input validation, calculations, and formatted output.

## Features
- Stores name, age, city, and hobby.
- Accepts favorite food and favorite color.
- Validates empty input.
- Calculates age in months.
- Displays information in a formatted layout.

## Requirements
- Python 3

## How to Run

```bash
python personal_info.py
```

## Sample Output

```text
Name           : Ranjith Kumar
Age            : 20
Age in Months  : 240
City           : Bangalore
Hobby          : Reading
Favorite Food  : Pizza
Favorite Color : Blue
```

## Test Cases
1. Valid input → Information displayed correctly.
2. Empty favorite food → Shows validation message.
3. Empty favorite color → Shows validation message.

- User Input/Output
- While Loops
